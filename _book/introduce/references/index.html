开发

# 克隆项目
git clone https://github.com/liuya6/h5-sdk-has-pwa

# 进入项目目录
cd h5-sdk-has-pwa

# 安装依赖
yarn

# 启动服务
yarn serve

发布

# 构建生产环境
yarn build
